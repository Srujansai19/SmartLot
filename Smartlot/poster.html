SmartLot: An IoT-Enabled Smart System with Machine Predictions

By Batch-106

Introduction

SmartLot addresses the common problem of finding available parking in urban areas. This project details an IoT-based system that detects vehicle presence in real-time.
It leverages machine learning to analyze historical data and predict future parking availability, providing users with accurate notifications and reducing traffic congestion.

Machine Learning

  1. Data Logging: All sensor status changes are logged with timestamps.
  2. Data Preprocessing: Data is cleaned and aggregated into hourly occupancy rates.
  3. Model Training: A Long Short-Term Memory (LSTM) network is trained on 30 days of data.
  4. Deployment: The model runs on the Flask server, re-training weekly.

Solution

SmartLot provides a complete, sensor-to-screen solution for smart parking management. By combining reliable IoT hardware with predictive machine learning, the system offers significant value to both parking facility operators and end-users. The system is scalable, cost-effective, and helps reduce urban traffic congestion and emissions.
